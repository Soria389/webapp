# 💰 WebAhorro - Control Inteligente de Finanzas Personales

## 📌 Descripción

WebAhorro es una aplicación web basada en una plantilla de ahorro en Excel que permite al usuario gestionar sus finanzas mensuales de forma sencilla, visual e intuitiva.

El usuario puede registrar cada mes sus ingresos previstos y reales, clasificar sus gastos por categorías y obtener automáticamente el balance total del mes, incluyendo cuánto ha gastado y cuánto ha conseguido ahorrar.

Además, la aplicación incluye un sistema comparativo entre meses anteriores para analizar la evolución financiera sin necesidad de cargar manualmente cada plantilla mensual.

---

## 🎯 Objetivo del Proyecto

Digitalizar y mejorar una plantilla tradicional de Excel para:

- Automatizar cálculos
- Visualizar balances mensuales
- Comparar meses anteriores
- Facilitar el análisis del ahorro
- Centralizar toda la información financiera en un único lugar

---

## 🚀 Funcionalidades Principales

### 📅 Gestión Mensual

- Añadir un nuevo mes
- Introducir:
  - Ingreso previsto
  - Ingreso real
- Registrar gastos organizados por categorías

---

### 🏷️ Categorías de Gastos

El usuario puede registrar gastos en distintas categorías como:

- Vivienda
- Alimentación
- Transporte
- Ocio
- Suscripciones
- Otros gastos personalizados

La aplicación calcula automáticamente:

- Total gastado
- Diferencia entre ingreso previsto y real
- Ahorro final del mes

---

### 📊 Balance Automático

El sistema calcula en tiempo real:

- Total de gastos
- Ahorro mensual
- Diferencia entre previsión e ingresos reales
- Porcentaje de ahorro

---

### 📈 Comparativa entre Meses

La aplicación permite:

- Visualizar todos los meses registrados
- Comparar ingresos, gastos y ahorro
- Analizar patrones de consumo
- Detectar meses con mayor gasto
- Evaluar evolución financiera

Todo esto sin necesidad de cargar manualmente las plantillas de cada mes.
